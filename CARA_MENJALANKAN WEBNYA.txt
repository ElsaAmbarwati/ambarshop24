# Cara Menjalankan Website AmbarShop24

1. Pastikan sudah install Node.js (https://nodejs.org/)
2. Buka folder project ini di terminal (PowerShell/Command Prompt)
3. Install dependencies:
   npm install
4. Jalankan website:
   npm run dev
5. Buka browser dan akses:
   http://localhost:5173

Jika ada error, pastikan Node.js versi terbaru dan sudah install semua dependency.

Catatan: Untuk produksi, gunakan perintah build dan preview:
   npm run build
   npm run preview
